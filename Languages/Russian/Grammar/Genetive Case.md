

## Definition
To show possession like in english ('s) or the word (of), also used to replace "some", "any", and negation, like there is not.

Ending same in plural animate nouns than accusative
Ending same in singular masculine animate nouns than accusative

## Singulier
### Masculine Nouns
- end consonant, add "a" 
- стол - стола
- й -> я 
- герой - героя
- ь -> я 
- конь коня
Exceptions дядя - дяди, папа - папы
### Femenine Nouns
- a -> ы
- страна - страны
- exception г к х ж ч ш щ + а -> и
- книги
- я -> и
- деревня - деревни
- ь -> и
- кровать - кровати
- ия -> ии
- лекция - лекции
### Neuter Nouns
- о -> a
- слово - слова
- е -> я
- поле - поля
- ие -> ия
- здание - здания
## Pluriel
### Masculine Nouns

- ж, ч, ш, щ, ь  add -> ей
- конь -> коней, этаж -> этажей, врач -> врачей, друг -> друзей 
- й, ц (stressed) add -> ев
- музей -> музеев, герой -> героев
- all other ов
- студент -> студентов, стол -> столов
### Femenine Nouns
- а -> drop a
- страна -> стран, книга -> книг, девушка -> девушек
- ь -> add ей
- consonant я -> ь
- деревня -> деревни
- vowel я -> й
- лек
 
### Neuter Nouns
- о -> drop о
- е -> ей
- ие -> ий
