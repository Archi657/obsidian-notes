## Definition

==IMPORTANT TO KNOW== [[Compabilities]]

If the adjective's stress is positioned on the final syllable, it will have the ending "ой" in the masculine nominate. If the adjective's stress falls on the stem, the masculine nominative ending will be "ый."

### Possible endings

| Gender | Ending      | Example 1 | Example 2 | Example 3 | Example 4 |
| ------ | ----------- | --------- | --------- | --------- | --------- |
| M      | -ый -óй -ий | большой   | синий     | больной   | маленький |
| F      | -ая -яя     | большая   | синяя     | больная   | маленькая |
| N      | -ое -ее     | большое   | синее     | больное   | маленькое |
| P      | -ые -ие     | большые   | синие     | больные   | маленькие |


### Hard Consonant  

| Gender    | Ending  |
| --------- | ------- |
| Masculine | -ой -ый |
| Neuter    | -ая     |
| Femenine  | -ое     |
| Plural    | -ые     |
### Soft Consonant  
| Gender    | Ending |
| --------- | ------ |
| Masculine | -ий    |
| Neuter    | -яя    |
| Femenine  | -ее    |
| Plural    | -ие    |
### Adjectives with stem ending in "г," "к," "х"
| Gender    | Ending  |
| --------- | ------- |
| Masculine | -ой -ий |
| Neuter    | -ая     |
| Femenine  | -ое     |
| Plural    | -ие     |
### Adjectives with stem ending in "ж," "ш," "ч," щ"

| Gender    | Ending  |
| --------- | ------- |
| Masculine | -ой -ий |
| Neuter    | -ая     |
| Femenine  | -ое -ее |
| Plural    | -ие     |
